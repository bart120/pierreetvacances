AKS - Jour 3

Démo GitLab CI/CD + Helm + Argo CD + Grafana

Cette application est une page statique servie par Nginx.

Elle sert de support pour montrer une chaîne DevOps simple vers AKS.

  • Build Docker avec GitLab CI/CD
  • Déploiement Kubernetes avec Helm
  • Synchronisation GitOps avec Argo CD
  • Supervision avec Prometheus et Grafana
